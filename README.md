# OD600_Transformer_Zihuan518

## 功能介绍

本工具可以将微孔板阅读器（资环楼518酶标仪）导出的Excel格式OD600数据转换为标准的CSV格式，便于后续的数据分析和绘图。

## 输入要求

- **文件格式**: Excel文件 (.xls 或 .xlsx)
- **数据内容**: 48小时细菌生长曲线OD600测定数据
- **数据布局**: 96孔板格式，每行代表一个孔位，每列代表一个时间点

## 输出格式

生成的CSV文件包含以下4列：

| 列名 | 描述 | 示例 |
|------|------|------|
| Well | 孔位编号 | A01, A02, B01, ... |
| Time_s | 时间（秒） | 0.0, 1800.0, 3600.0, ... |
| Time_h | 时间（小时） | 0.0, 0.5, 1.0, ... |
| OD | 光密度值 | 0.2850, 0.3120, ... |

## 使用方法

### 命令行运行

```bash
python3 bacterial_growth_converter.py "输入文件.xls" "输出文件.csv"
```

**示例:**
```bash
python3 bacterial_growth_converter.py "WZOD600.xls" "growth_data.csv"
```

## 输出示例

转换后的CSV文件内容示例：

```csv
Well,Time_s,Time_h,OD
A01,0.0,0.0,0.0996
A01,3600.3,1.0,0.0918
A01,7200.3,2.0,0.0916
A1,5400.0,1.5,0.3180
A2,0.0,0.0,0.2900
A2,1800.0,0.5,0.2970
...
```

## 数据特征

- **时间范围**: 0-48小时
- **时间间隔**: 通常为0.5小时（1800秒）
- **孔位数量**: 最多96个孔位（A1-H12）
- **OD值范围**: 通常在0-5之间

## 注意事项

1. **文件路径**: 如果文件路径包含空格或特殊字符，请用双引号包围
2. **数据质量**: 工具会自动过滤异常值（如负数或过大的OD值）
3. **时间检测**: 工具会自动检测Excel中的时间序列，如果检测失败会生成标准48小时序列
4. **孔位识别**: 工具会智能识别数据区域，跳过标题和空白行

## 故障排除

### 常见问题

1. **"ModuleNotFoundError: No module named 'pandas'"**
   - 解决方法: 安装pandas包 `pip install pandas`

2. **"文件不存在"错误**
   - 检查文件路径是否正确
   - 确保文件名拼写无误

3. **"未能提取到有效数据"错误**
   - 检查Excel文件格式是否正确
   - 确保数据区域包含数值型OD值

4. **数据点数量异常**
   - 检查Excel文件中的数据布局
   - 确认时间序列的完整性

### 获取帮助

如果遇到问题，可以：

1. 查看脚本运行时的详细输出信息
2. 检查输入Excel文件的格式和内容
3. 验证输出CSV文件的格式是否正确
